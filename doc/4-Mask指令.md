# 4. Mask 指令

本组包含以下指令：

- `tmmv`
- `tmnot`
- `tmand`
- `tmnand`
- `tmor`
- `tmnor`
- `tmxor`
- `tmnxor`
- `tmeq`
- `tmne`
- `tmlt`
- `tmge`
- `tmle`
- `tmgt`

## 1. 布尔视图约定

Mask 指令都把输入 tile 当作布尔矩阵看待：

- 零值视为假
- 非零值视为真
- 输出真写成 `1`
- 输出假写成 `0`

能被看作布尔视图的输入输出寄存器必须是 8 bit 宽类型的普通寄存器、`Temp` 或 `Mask`。

## 2. 布尔逻辑指令

### `tmmv`

- 布尔搬运
- 等价于把输入按“零/非零”归一化后写出

### `tmnot`

- 布尔非

### `tmand`

- 布尔与

### `tmnand`

- 布尔与非

### `tmor`

- 布尔或

### `tmnor`

- 布尔或非

### `tmxor`

- 布尔异或

### `tmnxor`

- 布尔同或

## 3. 比较指令

### `tmeq` / `tmne`

- 判断相等 / 不等
- 整数按整数值比较
- 浮点按浮点比较规则比较

### `tmlt` / `tmge` / `tmle` / `tmgt`

- 分别表示 `< / >= / <= / >`
- 输入两个操作数类型必须相同
- 支持所有整数与浮点类型

## 4. 操作数约束

### 布尔逻辑类

- `trdt_m8`：8 bit 宽普通寄存器、`Temp` 或 `Mask`
- `trs1zt_m8`：8 bit 宽普通寄存器、`0` 寄存器、`Temp` 或 `Mask`
- `trs2z_m8`：8 bit 宽普通寄存器、`0` 寄存器或 `Mask`

### 比较类

- 输出 `trdt_m8` 仍然必须是 8 bit 宽布尔结果寄存器
- 输入 `trs1zt` 和 `trs2z` 则使用原始数据类型，不要求是 8 bit

## 5. 与隐式 `TMASK1/2/3` 的关系

本组指令本身不读取当前 `TMASK1/2/3` 作为执行条件。

但是：

- 若输出目标就是 `TMASK1/TMASK2/TMASK3`
- 则这些指令可以直接生成后续其他指令要用的掩码 tile
