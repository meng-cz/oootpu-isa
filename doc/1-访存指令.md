# 1. 访存指令

本组包含以下指令：

- `tload`
- `tstore`
- `tloadRow`
- `tstoreRow`
- `tgatherRow`
- `tscatterRow`
- `tim2cfgNHWC`
- `tim2col`

## 通用约定

- tile 逻辑大小固定为 `32 x 32`
- 地址类型为 `TMemAddr`
- 地址必须按 `元素字节数 x 32` 对齐
- `rowstride` 单位是“元素数”，不是字节数
- `mask` 参数类型为 `uint8_t`
- `mask = 0`：不使用掩码
- `mask = 1/2/3`：分别使用隐式 `TMASK1/TMASK2/TMASK3`

## 1. `tload`

```cpp
void tload(const TOPRand &trd, TMemAddr baseaddr, uint8_t mask = 0);
```

语义：

- 从连续内存读取一个完整 tile 到 `trd`
- 等价于 `tloadRow(trd, baseaddr, TILE_SIZE, mask)`

掩码行为：

- `mask = 0`：读取全部 32x32 元素
- `mask = 1/2/3`：仅对对应 `TMASKx` 为真的位置执行内存读取，其他位置写入 `0`

## 2. `tstore`

```cpp
void tstore(const TOPRand &trs1z, TMemAddr baseaddr, uint8_t mask = 0);
```

语义：

- 将 `trs1z` 的 tile 内容按连续布局写回内存
- 等价于 `tstoreRow(trs1z, baseaddr, TILE_SIZE, mask)`

掩码行为：

- `mask = 0`：写出全部元素
- `mask = 1/2/3`：仅对对应 `TMASKx` 为真的位置执行内存写入

## 3. `tloadRow`

```cpp
void tloadRow(const TOPRand &trd, TMemAddr baseaddr, uint32_t rowstride, uint8_t mask = 0);
```

语义：

- 按行主布局加载一个 tile
- 第 `i` 行起始地址为 `baseaddr + i * rowstride * elem_size`
- 每行读取 `32` 个元素

适用场景：

- 从带 pitch 的二维内存区域取一个 `32 x 32` tile

## 4. `tstoreRow`

```cpp
void tstoreRow(const TOPRand &trs1z, TMemAddr baseaddr, uint32_t rowstride, uint8_t mask = 0);
```

语义：

- 按行主布局把 tile 写回内存
- 第 `i` 行写回地址为 `baseaddr + i * rowstride * elem_size`

## 5. `tgatherRow`

```cpp
void tgatherRow(const TOPRand &trd, const TOPRand &trs2_i, TMemAddr baseaddr, uint8_t rowsel, uint8_t mask = 0);
```

语义：

- 从 `trs2_i` 的第 `rowsel` 行取出 32 个整数偏移
- 这些偏移的单位是“元素”，不是字节
- 对输出 tile 的每一行 `i`：
  - 先取偏移 `offset = trs2_i[rowsel][i]`
  - 再从 `baseaddr + offset * elem_size` 读取连续 `32` 个元素
  - 作为输出 tile 的第 `i` 行

索引类型：

- `trs2_i` 必须是整数类型
- 支持 `8/16/32` bit
- 有符号整数按有符号扩展，无符号整数按无符号解释

掩码行为：

- `mask = 0`：32 行都参与
- `mask = 1/2/3`：检查 `TMASKx[rowsel][i]`
- 若该掩码位为假，则输出 tile 的第 `i` 行全部置 `0`
- 若该掩码位为真，则整行从内存加载

## 6. `tscatterRow`

```cpp
void tscatterRow(const TOPRand &trs1z, const TOPRand &trs2_i, TMemAddr baseaddr, uint8_t rowsel, uint8_t mask = 0);
```

语义：

- 与 `tgatherRow` 相反
- 从 `trs2_i` 的第 `rowsel` 行取 32 个整数偏移
- 将 `trs1z` 的第 `i` 行写到 `baseaddr + offset * elem_size` 开始的连续 32 个元素位置

掩码行为：

- `mask = 0`：写出全部 32 行
- `mask = 1/2/3`：仅当 `TMASKx[rowsel][i]` 为真时，才写出第 `i` 行

## 7. `tim2cfgNHWC`

```cpp
void tim2cfgNHWC(Im2colLayout layout, TMemAddr baseaddr, uint16_t N, uint16_t H, uint16_t W, uint16_t C);
```

语义：

- 配置后续 `tim2col` 使用的 NHWC 输入张量描述符
- 输入张量逻辑形状为 `N x H x W x C`
- `baseaddr` 指向第 `0` 个元素，按 NHWC 连续存放
- 输出的 im2col 逻辑矩阵布局为 `N*OH*OW` 行，`KH*KW*C` 列

布局参数：

- `K1S1P0`：`kernel=1, stride=1, padding=0`，`OH=H, OW=W`
- `K3S1P1`：`kernel=3, stride=1, padding=1`，`OH=H, OW=W`
- `K3S2P1`：`kernel=3, stride=2, padding=1`，`OH=(H+1)/2, OW=(W+1)/2`
- `K5S1P2`：`kernel=5, stride=1, padding=2`，`OH=H, OW=W`
- `K7S2P3`：`kernel=7, stride=2, padding=3`，`OH=(H+1)/2, OW=(W+1)/2`

地址约束：

- `baseaddr` 必须按 `元素字节数 x 32` 对齐

## 8. `tim2col`

```cpp
void tim2col(const TOPRand &trd, uint16_t subtile_row, uint16_t subtile_col);
```

语义：

- 使用最近一次 `tim2cfgNHWC` 配置的描述符，从 NHWC 输入张量中按 im2col 规则加载一个 `32 x 32` tile 到 `trd`
- 整个 im2col 结果矩阵的行数为 `N*OH*OW`，列数为 `KH*KW*C`
- `subtile_row` 和 `subtile_col` 分别指定该逻辑矩阵的行子 tile 索引和列子 tile 索引
- tile 内第 `i, j` 个元素对应逻辑矩阵中的：
  - 行 `row = subtile_row * 32 + i`
  - 列 `col = subtile_col * 32 + j`

行列映射：

- 行索引先映射到 `(n, oh, ow)`：
  - `n = row / (OH*OW)`
  - `ohow = row % (OH*OW)`
  - `oh = ohow / OW`
  - `ow = ohow % OW`
- 列索引映射到卷积窗口和通道 `(kh, kw, c)`：
  - `c = col % C`
  - `k = col / C`
  - `kh = k / KH`
  - `kw = k % KH`

输入坐标：

- 对应输入位置为：
  - `ih = oh * stride - padding + kh`
  - `iw = ow * stride - padding + kw`
- 读取的是 `input[n][ih][iw][c]`

越界行为：

- 若 `row` 超出 `N*OH*OW`，输出为 `0`
- 若 `col` 超出 `KH*KW*C`，输出为 `0`
- 若 `(ih, iw)` 落在 padding 区域或超出输入边界，输出为 `0`

操作数约束：

- `trd` 只能是普通寄存器

## 9. 操作数约束

- `tload/tloadRow/tgatherRow` 的目标 `trd` 只能是普通寄存器
- `tstore/tstoreRow/tscatterRow` 的源 `trs1z` 可为普通寄存器或 `0` 寄存器
- `tgatherRow/tscatterRow` 的索引寄存器 `trs2_i` 必须是整数类型普通寄存器
- `tim2col` 的目标 `trd` 只能是普通寄存器

## 10. 实现补充

- 内存按小端字节序组装和拆解 `16/32` bit 元素
- 支持元素宽度仅 `1/2/4` 字节
